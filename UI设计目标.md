# 竹林司马 UI 目标文档

> **目标**：打造科技感强烈、数据展示简洁直接的下一代量化分析界面

---

## 一、核心设计原则

### 1. 科技动态效果（Tech-Dynamic）
让界面"活起来"，通过微妙的动效传达数据的生命力和专业感。

### 2. 简洁直接（Simple-Direct）
信息密度高，但认知负担低。一眼扫过就能抓住关键。

---

## 二、科技动态效果目标

### 2.1 背景与氛围

| 效果 | 现状 | 目标 | 优先级 |

## 三、数据展示简化目标

### 3.1 StockCard 简化

**现状问题**：
- 展示代码（如 600406.SH）— 用户不需要看代码
- 成交额 label — 大部分用户不需要

**目标布局**：

```
┌─────────────────────────────────┐
│  股票名称            当前价格   │  ← 名称大字，右侧价格
│  ▲ +2.35%              ¥28.65  │  ← 涨跌一行，简洁
│  ▁▂▃▅▆▇走势迷你图 ▇▆▅▃▂▁    │  ← 迷你趋势线
└─────────────────────────────────┘
```

**精简字段**：
- ❌ 删除：股票代码（tsCode）
- ❌ 删除："成交额" label + 成交额数值
- ✅ 保留：名称、价格、涨跌幅、迷你趋势图

### 3.2 PriceTicker 详情区简化

**现状问题**：
- 昨收 "--" 始终无数据，却占位显示
- 技术指标 MA5/MA20/MA60/EMA12 全英文缩写混排
- 开盘/最高/最低/成交量/成交额 6项一股脑排开

**目标布局**：

```
┌──────────────────────────────────────────────────────────────┐
│  国电南瑞 · 600406                                         │  ← 名称 + 代码（一行）
│  28.65  ▲ +0.65 (+2.32%)                                  │  ← 价格 + 涨跌（一行大字）
│  ─────────────────────────────────────────────────────────  │
│  今开 28.10    最高 29.10    成交 12.3万                    │  ← 一行：开盘/最高/成交量
│  ─────────────────────────────────────────────────────────  │
│  MA5 28.42  MA20 27.80  MA60 26.50   RSI 68 ▲              │  ← 一行：均线 + RSI
└──────────────────────────────────────────────────────────────┘
```

**精简规则**：
- ❌ 删除："昨收"（始终为"--"）
- ❌ 删除："成交额"（换成更大字号的"成交量"）
- ✅ 均线标签改为 `MA5/20/60` 更紧凑写法
- ✅ RSI 状态颜色直接融入（>70红色 <30绿色）
- ✅ 所有信息压缩到 **5行以内**

### 3.3 预警面板简化

**现状问题**：
- 预警信息冗长，含大量文字说明

**目标**：每条预警一行 + 图标，颜色表示等级。

## 五、技术方案

### 动画库
- **Framer Motion**（已安装 v11）：复杂路由动画、布局动画
- **Tailwind CSS**：基础悬停/过渡动画
- **CSS Keyframes**：背景、脉冲等持续性动画

### 新增工具函数
```typescript
// src/lib/animation.ts
export const cn = ... // 已有
export const springTransition = { type: "spring", stiffness: 300, damping: 30 }
export const easeOut = "cubic-bezier(0.16, 1, 0.3, 1)"
```

### 动画变量令牌
```css
/* index.css */
:root {
  --pulse-duration: 2s;
  --glow-rise: 0 0 20px rgba(239, 68, 68, 0.3);
  --glow-fall: 0 0 20px rgba(34, 197, 94, 0.3);
  --spring: cubic-bezier(0.34, 1.56, 0.64, 1);
}
```
